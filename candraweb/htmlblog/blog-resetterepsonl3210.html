Cara Mengatasi Aplikasi Resetter Epson L3210 yang Diblokir Antivirus di Windows 11

Diposting pada 24 Mei 2025 oleh Candra

Kadang kita butuh banget pakai aplikasi resetter untuk printer Epson, tapi malah kena drama: "Aplikasi langsung dikarantina oleh Windows Defender!" Tenang, kamu nggak sendirian kok. Windows 11 emang kadang terlalu overprotective, terutama kalau aplikasi nggak punya digital signature.

Nah, buat kamu yang mau pakai resetter Epson L3210 tanpa diblokir antivirus, ini dia langkah-langkah simpel dan aman:

1. Matikan Koneksi Internet

Kenapa? Karena antivirus aktif otomatis saat online. Biar lebih aman dan proses instalasi lancar, putuskan dulu koneksi internet.


      Kalau pakai WiFi: klik ikon WiFi di taskbar → pilih Disconnect.
      Kalau pakai kabel LAN: cabut kabelnya dulu.
      Kalau laptop kamu support Airplane Mode, aktifkan saja sebentar.

2. Nonaktifkan Antivirus (Windows Defender)

Buka Settings → Privacy & Security → Windows Security → Virus & Threat Protection → Manage settings, lalu matikan Real‑time protection.

3. Tambahkan Folder Resetter ke Exclusion List

Tambahkan folder tempat resetter berada ke daftar pengecualian agar antivirus tidak otomatis menghapus file saat re‑aktif.

Masih di Windows Security, klik Manage settings lagi.

Scroll ke bawah → Add or remove exclusions.
Klik Add an exclusion → pilih Folder.

Pilih folder tempat resetter disimpan.

Catatan: Jangan pindahkan folder resetter ke drive C:\Program Files, simpan saja di folder biasa seperti D:\Resetter atau Documents

4. Ekstrak & Jalankan Resetter

Ekstrak dari .zip/.rar lalu jalankan adjprog.exe dengan klik kanan → Run as administrator.

5. Pilih Mode Reset dan Lakukan Reset

Pilih printer L3210, klik Particular Adjustment Mode → Waste Ink Pad Counter. Centang lalu klik Check → Initialize, tunggu proses selesai.

6. Matikan, Nyalakan Ulang Printer, dan Aktifkan Antivirus + Internet

Setelah reset selesai, restart printer. Lalu aktifkan kembali antivirus dan sambungkan internet.

Catatan Tambahan

Resetter Epson L3210 ini bukan virus, tapi karena tidak dikenal sistem, sering dikira berbahaya. Dengan cara di atas, kamu bisa pakai aplikasinya tanpa hambatan, dan tetap aman selama tahu sumber download-nya terpercaya.
