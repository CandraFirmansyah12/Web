Konfigurasi Mikrotik di Winbox - Job Sheet 2

Diposting pada 16 Agustus 2022 oleh Candra

Sebelum Mulai: Pastikan Selesai Job Sheet 1

Penjelasan: Ini penting untuk memastikan Mikrotik sudah terhubung internet dan siap diuji lanjutan.

1. Ubah Identitas MikroTik

Penjelasan: Mengganti nama perangkat memudahkan pengenalan router, apalagi jika punya banyak perangkat serupa.

Akses System → Identity, masukkan nama perangkat baru, lalu klik Apply → OK.

2. Atur Waktu Lokal

Penjelasan: Waktu lokal diperlukan untuk sinkronisasi log, jadwal, dan fitur keamanan lainnya.

Masuk ke System → Clock, ubah jam, tanggal, dan zona ke Asia/Jakarta.

3. Aktifkan SNTP Client

Penjelasan: SNTP auto sinkron jam dari server, menjaga waktu router tetap akurat.

Menu: System → SNTP Client. Aktifkan client dan isi Server dengan 203.160.128.3, lalu Apply.

Uji Coba Konfigurasi

    Cek Tabel ARP

    Penjelasan: ARP menunjukkan device yang tersambung, penting untuk deteksi IP dan MAC di jaringan lokal.

  • ARP: Akses IP → ARP untuk cek IP-MAC devices.

    • Jalankan IP Scan

    Penjelasan: IP Scan memudahkan melihat rentang IP aktif di interface tanpa pengecekan manual.

  • IP Scan: Di Tool → IP Scan, pilih interface dan jalankan scan.

    • Lakukan MAC Scan

    Penjelasan: MAC Scan membantu data MAC address device yang tidak terlihat di IP Scan.

  • MAC Scan: Jalankan tool mac-scan ether2 di terminal.

    • Gunakan Torch

    Penjelasan: Torch memberi info trafik real-time, membantu analisa penggunaan bandwidth dan port aktif.

  • Torch: Akses Tools → Torch, pilih ether2, lalu klik Start.

Keterangan Tambahan

  • Pastikan koneksi Job Sheet 1 berhasil sebelum lanjut agar tidak terjadi kesalahan konfigurasi.
  • Urutnya harus benar—mengganti identitas, sinkron jam, baru tools monitoring biar data akurat.
  • Cek hasil tiap langkah langsung biar tau ada masalah sejak awal.
